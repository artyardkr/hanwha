유튜브, 크리에이터 수익 정산에 페이팔 스테이블코인 ‘PYUSD’ 도입

📺 유튜브, PYUSD 정산 도입
- 유튜브가 크리에이터 수익 정산 수단으로 페이팔의 스테이블코인 ‘PYUSD’를 추가
- 일정 자격을 갖춘 미국 내 유튜버들은 수익금을 PYUSD로 정산받을지 선택할 수 있게됨
- 유튜버가 조회수 수익, 채널 멤버십, 슈퍼챗, 슈퍼땡스 등으로 수익을 창출하면, 이 수익금을 기존처럼 달러로 받을지 PYUSD로 받을지 선택할 수 있는것
🪙 PYUSD 정산 방식
- 유튜브가 수익금을 크리에이터에게 직접 송금하지 않음
- 유튜브는 주로 페이팔의 하이퍼월렛(Hyperwallet) 인프라를 통해 지급 처리업체로 수익금을 전달함
- 즉, 유튜브가 PYUSD를 직접 보유하거나 관리하지 않아도 되는것
- 페이팔이 중간에서 달러 → PYUSD 전환과 분배를 담당하는 방식
- 크리에이터가 정산 방식을 PYUSD로 선택하면, 페이팔이 정산금을 PYUSD로 바꿔서 입금해주면 되기 때문에 과정 자체가 복잡한 편은 아님
👛 유튜버의 PYUSD 활용법
- 유튜버가 PYUSD로 수익금을 정산 받으면 페이팔 생태계 내에 그대로 보유하거나, 필요시 다시 미 달러로 환전 가능
- 또한 PYUSD를 지원하는 블록체인 네트워크를 통해 외부 지갑으로 전송하는 것도 가능
- PYUSD로 정산을 받는다고 해서 엄청난 혜택이 있는것은 아님
- 단순히 달러 지급 방식에서 스테이블코인으로 바뀌는 것
✍️ PYUSD 정산 방식의 장점과 단점
장점 : 기존 은행 송금은 영업일, 마감 시간, 공휴일 등에 영향을 받아 입금이 지연될 수 있음. 반면, 스테이블코인은 24시간 365일 가동되므로 정산 처리 즉시 빠르게 자금을 수령할 수 있음
단점 : PYUSD를 다시 현금(달러)으로 환전하거나 외부로 보낼 때 수수료가 발생할 수 있음. 또한, 암호화폐 수취에 따른 회계 처리 및 세무 신고 절차가 기존보다 복잡해질 가능성이 있음
👋 업계 반응
- 업계에서는 유튜브 정산 옵션에 PYUSD가 들어온 것을 대체로 긍정적으로 평가하는 분위기
- 스테이블코인이 단순한 투자 상품을 넘어 달러를 이동시키는 대안적인 네트워크로서 주류 결제 시스템으로 자리잡기 시작했다고 설명
